A web chatbot trained using movie quotes!

Usage example:

  • GET: /question/how are you?
    • Browser handles spaces (/question/how%20are%20you?)
    • Returns raw response (ex: I am good).
  • POST: /admin/retrain/ body/JSON: {"conversation_length": int, "stop_short": int}:
    • Retrain's chatbot with parameters:
      • conversation_length - the minimum number of statements required in each conversation
      • stop_short - a maximum limit of how many conversations to train on
    • Returns response of training (ex: Retrained with parameters: {'conversation_length': 5, 'stop_short': 10000}".
